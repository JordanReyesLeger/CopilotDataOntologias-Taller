Modo Agent: Crear archivos automáticamente

Cambiamos a Agent Mode (🤖) y le pedimos que genere la ontología desde el CSV.

Prompt en Agent Mode
🤖 Lee data/censo_2020_sample.csv.
Genera ontology/censo_ontology.json
con clases, propiedades y reglas
de calidad.
El agente ejecuta
Paso 1: Leyendo CSV...
Paso 2: Creando ontology/censo_ontology.json
Paso 3: Escribiendo 3 clases, 11 propiedades
Paso 4: ✓ Archivo creado

▶ Acepta o rechaza los cambios

💡 La diferencia con Ask: el agente lee, crea y escribe archivos. No solo responde.

Crear un Agente Personalizado

Un archivo .agent.md en .github/agents/ crea un asistente especializado.

El archivo que creamos:

--- description: "Analista censal INEGI" tools: ["codebase", "search"] --- # Analista de Datos Censales Eres un DBA senior del INEGI. SIEMPRE lee: - ontology/censo_ontology.json - data/censo_2020_sample.csv Responde en español. Usa nombres semánticos de la ontología. Reporta problemas de calidad.

Después de recargar VS Code:

Agente: Analista Censales INEGI

Agent Mode vs Coding Agent

💻 Agent Mode (en tu IDE)

  • 🟢 Trabaja dentro de VS Code
  • 🟢 Colaborador en tiempo real
  • 🟢 Itera sobre código y corre tests
  • 🟢 Tú diriges, él ejecuta

🌐 Coding Agent (en GitHub)

  • 🟠 Toma issues asignados
  • 🟠 Escribe código y pasa tests
  • 🟠 Abre PRs automáticamente
  • 🟠 Compañero de equipo IA

📄 AGENTS.md — Nuevo formato soportado en VS Code. Un archivo de instrucciones para todos tus agentes. Compatible con copilot-instructions.md, CLAUDE.md y GEMINI.md.

¿Por qué ontologías para INEGI?

01

Vocabulario Común

Un lenguaje compartido entre Economía, SocioDemografía, Gobierno y Geografía.

02

Significado Explícito

El LLM entiende “población económicamente activa” en lugar de pea_15_mas.

03

Inferencia Automática

La ontología deduce: “todo municipio pertenece a una entidad federativa”.

04

Interoperabilidad

Estándares W3C (RDF, OWL, SKOS). Cualquier consumidor de IA lee los datos.

05

AI-Ready

El LLM consume la ontología como contexto y responde con precisión semántica.
